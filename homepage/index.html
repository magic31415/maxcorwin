Projects

  • What happened under Trump?: A website to see all the bills that have been enacted by the federal government since President Trump took office. Although this was originally intended to be a updated version of this website to reflect the changes under the new president, I have started adding additional features such as insight on how both chambers of congress, as well as your representatives, voted on each bill. This project is just getting started and there will be many more features to come! Having said that, this is very much a work in progress, so not everything is working perfectly yet. The stack consists of Phoenix, Elixir, Postgres, and ReactJS. Most of the data comes from the ProPublica Congress API, but the ability to get your district by your address comes from the Google Civic Information API. Github Repository
  • What's Hoppin: A website for beer enthusiasts to browse for and learn about different types of beer. Users can post comments on beer style and brewery pages. Stack consists of Phoenix, Elixir, Postgres, and Nginx. Data from BreweryDB API. Github Repository
  • Microblog: A Twitter-like website where users can post messages, like messages, and follow other users. Github Repository
